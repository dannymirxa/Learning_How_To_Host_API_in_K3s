# learning_how_to_host_api_in_k3s

Tools used:

1. Bun
2. Express
3. DuckDB
4. Zod

Objectives of this Project:

1. Create a working RESTfull API with Bun.
2. Deploy it to K3S.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run server.ts
```

To open Employees.db with DuckDB CLI:

```bash
duckdb Employee.db
```
